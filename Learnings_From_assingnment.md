### What I Learned from this assignment 

The gave assignment help me understand and build a practical application using Django to solve real world warehouse problem. The mail goal of the project was to recommend a suitable box for the product based on the dimensions of the product.

While working on this project, I focused on how the different parts of Django applications work together, including database models, views, forms, APIs, and testing.

### Django Concepts I learned 
•	Models and database relationships
•	Views
•	Forms
•	URL routing
•	Templates
•	Django REST Framework serializers
•	Management commands
•	Unit testing

### Recommendation Logic
•	Calculating total weight
•	Calculating total volume
•	Filtering eligible boxes
•	Selecting the lowest-cost valid box
•	Handling cases where no suitable box exists

### Conclusion 
In this project Beyond writing the code, I learned how to break real world problem into smaller components such as designing database models, implementing business logic, building APIs, creating a simple user interface, and testing the final application.

One of the most important part I learned was the debugging, understanding the error messages and testing different scenarios helped me solve isues. Overall, this project strengthened both my technical skills and my confidence in building practical backend applications using Django, while showing me that careful testing and continuous improvement are just as important as implementing new features.
